### Windows

Copy `header-cli` to a location of your choice. Then, create a batch file to call it from the terminal:

e.g. `header-cli.bat`:
```cmd
@echo off
python C:\path\to\header-cli %*
```
Then, copy `header-cli.bat` to a location in your PATH.
